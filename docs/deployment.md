# Deployment guide (free tiers)

1. **Database:** Provision Neon PostgreSQL.
2. **Redis:** Provision Upstash Redis.
3. **Search:** Deploy Meilisearch on Railway.
4. **Backend:** Deploy NestJS API on Railway with env vars.
5. **Frontend:** Deploy Next.js on Vercel.
6. **Files:** Configure Cloudinary credentials.
7. **Monitoring:** Attach Sentry DSN + PostHog key.
8. **Email:** Configure Resend API key.
